# Yandex Station Socket

## Paths
- /play - Play music
- /stop - Stop music
- /next - Next track
- /prev - Previous track
- /rewind/100 - Rewind track (Put your value)
- /volume/0.2 - Set volume (Put your value)

## Install
- pip install -r requirements.txt
